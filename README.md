# Curso de HTML — De cero a una base profesional de estructura web

Repositorio oficial: <https://github.com/yetsin7/Curso-de-HTML>

## Documentacion de referencia

Consulta [Doc-HTML.md](./Doc-HTML.md) para estudiar HTML con enlaces oficiales
y una ruta clara de temas.

## Para que sirve este curso

Este curso esta pensado para que HTML deje de ser solo una introduccion y pase a
ser una base real para construir paginas, landings, catalogos, formularios y la
estructura inicial de interfaces mas grandes.

## Modulos del curso

1. `01-introduccion-y-estructura`
2. `02-texto-listas-y-enlaces`
3. `03-imagenes-tablas-y-embebidos`
4. `04-formularios-completos`
5. `05-html-semantico-y-accesibilidad`
6. `06-proyecto-final-html`
7. `07-seo-y-metadatos-html`
8. `08-multimedia-y-embed-avanzado`
9. `09-estructuras-de-blog-y-noticias`
10. `10-paginas-de-venta-y-landing`
11. `11-componentes-estaticos-de-ecommerce`
12. `12-proyecto-final-html-pro`

## Resultado esperado

Al terminar deberias poder estructurar:

- paginas informativas;
- landings;
- catalogos;
- paginas de producto;
- formularios;
- una mini tienda estatica base.

## Que estudiar despues

Si quieres seguir la ruta natural para frontend moderno, despues de HTML lo mas
recomendable es estudiar:

1. CSS: <https://github.com/yetsin7/Curso-de-CSS>
2. JavaScript: <https://github.com/yetsin7/Curso-de-JavaScript>
3. Vue y Nuxt: <https://github.com/yetsin7/Curso-de-Nuxt-Vue>
