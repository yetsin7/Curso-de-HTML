# Documentacion de HTML

HTML significa HyperText Markup Language, o lenguaje de marcado de hipertexto.
Es la base estructural de una pagina web: define que partes son titulos,
parrafos, imagenes, enlaces, formularios y secciones.

## Documentacion recomendada

- [MDN HTML Docs](https://developer.mozilla.org/en-US/docs/Web/HTML)
  MDN explica HTML de forma practica, con ejemplos y definiciones faciles de
  seguir.
- [WHATWG HTML Living Standard](https://html.spec.whatwg.org/)
  Es el estandar tecnico oficial moderno de HTML. Aqui se define como funciona
  realmente el lenguaje.

## Temas principales que debes estudiar

1. Estructura basica de un documento HTML:
   entender `<!DOCTYPE html>`, `html`, `head` y `body`.
2. Etiquetas principales:
   aprender las piezas mas usadas del lenguaje.
3. Encabezados y parrafos:
   organizar el contenido para que sea legible.
4. Enlaces e imagenes:
   conectar paginas y mostrar recursos visuales.
5. Listas y tablas:
   presentar informacion agrupada o tabular.
6. Formularios e inputs:
   capturar datos del usuario.
7. HTML semantico:
   usar etiquetas como `header`, `main`, `section` y `footer` para dar mejor
   significado al contenido.
8. Multimedia:
   integrar audio, video e iframes.
9. Accesibilidad basica:
   escribir HTML que tambien puedan usar personas con lectores de pantalla o
   navegacion por teclado.
10. SEO basico con HTML:
   ayudar a que buscadores entiendan mejor la pagina.

## Orden recomendado

1. [Estructura basica](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Structuring_content/Basic_HTML_syntax)
   Aqui aprendes como nace un documento HTML desde cero.
2. [Elementos HTML](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements)
   Esta referencia te muestra que hace cada etiqueta.
3. [Formularios](https://developer.mozilla.org/en-US/docs/Learn_web_development/Extensions/Forms)
   Te ensena a pedir datos al usuario de forma correcta.
4. [Semantica](https://developer.mozilla.org/en-US/docs/Glossary/Semantics)
   Te ayuda a escribir HTML con mejor significado y mejor estructura.

## Ejemplo basico

```html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi primera pagina</title>
</head>
<body>
  <h1>Hola mundo</h1>
  <p>Esta es mi primera pagina con HTML.</p>
</body>
</html>
```
